# 求人マッチングツール改修計画

## 概要
ブックマークレットを使って circus / HITO-Link から求人データを自動抽出し、
Streamlitアプリに貼り付けるだけでAIマッチングが完了するワークフローを実現する。

---

## 改修後のユーザーフロー

```
【初回セットアップ（1回だけ）】
1. アプリの「セットアップ」ページを開く
2. 「circus用」「HITO-Link用」のブックマークレットをブックマークバーにドラッグ

【毎回の操作】
1. アプリにログイン → 候補者を選択 or 登録
2. circus / HITO-Linkで検索条件を設定して検索
3. ブックマーク「🎯求人抽出」をクリック → 自動で全ページ抽出
4. アプリの「求人取込」ページで Ctrl+V → AIマッチ実行
5. 結果を確認 → 良い求人のPDFをダウンロード
6. PDFをアプリにアップロード → 精密採点
7. 推薦リストをExcelダウンロード
```

## 実装タスク

### 1. circus用ブックマークレット作成
**ファイル:** `bookmarklets/circus_extract.js`

- プロンプト集のReact Fiber抽出ロジックをそのまま使用
- 処理フロー:
  1. 画面にオーバーレイを表示（「抽出中...」）
  2. React Fiberからカード要素を取得
  3. `__extractPage()` で現在ページの全25件を抽出
  4. 次ページボタンをクリック → 2秒待機 → 繰り返し
  5. 全ページ完了後、JSON形式でクリップボードにコピー
  6. 「✅ 125件取得完了！」と表示
- circusはSPA（React）なのでページ遷移してもスクリプトが消えない
- JSONフォーマット:
  ```json
  {
    "source": "circus",
    "extracted_at": "2026-03-09T15:30:00",
    "jobs": [
      {
        "company": "HENNGE株式会社",
        "title": "セールス",
        "job_type": "シェアリング",
        "salary": "400万〜600万",
        "location": "東京都",
        "skills": "法人営業経験2年以上...",
        "published_at": "2026-01-15",
        "agent_company": "株式会社〇〇"
      }
    ]
  }
  ```

### 2. HITO-Link用ブックマークレット作成
**ファイル:** `bookmarklets/hitolink_extract.js`

- プロンプト集のテキスト抽出ロジックを使用
- 処理フロー:
  1. 画面にオーバーレイを表示
  2. `document.body.innerText` でページ全体のテキストを取得
  3. テキストから求人情報をパース（企業名、タイトル、年収、勤務地等）
  4. 次ページボタンをクリック → 2秒待機 → 繰り返し
  5. 全ページ完了後、JSON形式でクリップボードにコピー
  6. 完了メッセージ表示
- JSONフォーマット: circusと同じ構造（sourceが"hito-link"）

### 3. Streamlitアプリ改修

#### 3-1. 新ページ「🔧 セットアップ」追加
**ファイル:** `app.py`（サイドバーにメニュー追加）

- ブックマークレットの設置方法を説明
- ブックマークレットのリンクを表示（ユーザーがブックマークバーにドラッグ）
- ステップバイステップの説明文

#### 3-2. 「CSV取込 & マッチ」ページの改修
**ファイル:** `app.py`

- 入力モード選択を追加:
  - 「📋 ブックマークレットから貼り付け」（デフォルト・推奨）
  - 「📁 CSVファイルをアップロード」（従来方式）
- ブックマークレットモード:
  - 大きなテキストエリア（「ここにCtrl+Vで貼り付け」）
  - JSON自動検出 → パース → 求人一覧表示
  - 以降の流れ（AIマッチ・ダウンロード等）は同じ

#### 3-3. JSONパーサー追加
**ファイル:** `utils/csv_parser.py`

- 新関数: `parse_bookmarklet_json(json_text: str) -> list[dict]`
- circus/HITO-Link両方のJSONフォーマットに対応
- 既存の内部データ構造（jobs list）に変換

### 4. 画像付き操作マニュアル（Word）
**ファイル:** `求人マッチングツール_操作マニュアル.docx`（上書き）

- 各画面のスクリーンショット付き

## 注意事項
- bookmarkletsフォルダのJSはStreamlitアプリには含めない（ブラウザのブックマークとして配布）
- ブックマークレットはcircus/HITO-Linkのサイト構造変更で動かなくなる可能性がある
  → トラブルシューティング手順をマニュアルに記載
- 利用規約: DOM読み取りのみ、新規HTTPリクエストなし → 問題なし
